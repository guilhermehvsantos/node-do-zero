# Node do Zero

Aula da RocketSeat de desenvolvimento de uma aplicação CRUD com Node.js

## Aprendizado

Aula prática de backend, na qual pude aprender a base do Node.js integrando um banco de dados sql do zero ao deploy.

## Tecnologias

- Node.js
- Postgres
- Fastify
- Neon
- Render
